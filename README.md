# 四柱八字排盘系统 🔮

专业的四柱八字排盘网站应用，包含流年流月流日分析、神煞详批、刑冲合害等功能。

## ✨ 主要功能

### 1. 四柱八字显示
- 年柱、月柱、日柱、时柱完整展示
- 五行着色（金、木、水、火、土）
- 显示每个天干地支对应的五行属性
- 大运信息展示

### 2. 流年流月流日查询
- 可选择任意年份和月份查看流日
- 每个流日显示完整的公历日期和农历日期
- 显示星期信息
- 建除十二神（建、除、满、平、定、执、破、危、成、收、开、闭）

### 3. 天干地支交互分析
- **天干留意**：天干五合、天干相冲
- **地支留意**：地支六合、地支相冲、地支相刑、地支相害、地支相破
- 与原局四柱的交互关系一目了然

### 4. 30+种神煞系统

包含盲派及传统神煞：

## 🎨 界面特色

- 现代化渐变背景设计
- 五行颜色区分清晰直观
- 交互标签（合、冲、刑、害、破）用不同颜色标识
- 响应式设计，支持手机和电脑浏览
- 流畅的悬停效果增强用户体验

## 🚀 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 输入出生日期（公历，支持时辰选择）
3. 选择要查看的流年和流月
4. 点击"🔍 开始排盘"按钮
5. 查看详细的流日分析结果

## 📦 技术栈

- HTML5
- CSS3 + Tailwind CSS（通过 CDN）
- JavaScript（原生）
- lunar-javascript（农历和八字计算库）

## 🔧 本地运行

无需任何安装和配置，直接在浏览器中打开 `index.html` 即可使用。

在 macOS 上：
```bash
open index.html
```

在 Windows 上：
```bash
start index.html
```

或者直接双击文件，选择浏览器打开。
